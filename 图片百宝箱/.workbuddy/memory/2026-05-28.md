## 图片百宝箱 - 修复 Vercel 构建失败（Root Directory 配置错误）

- **问题**：Vercel 部署显示红色 X 失败，线上始终运行旧版单页 HTML
- **根因**：Vercel 项目的 **Root Directory** 被错误设置为 `"图片百宝箱"`，但仓库根目录下不存在此文件夹
  - 构建日志：`The specified Root Directory "图片百宝箱" does not exist. Please update your Project Settings.`
  - 每次构建直接失败，Vercel 自动回退到 2 周前的旧版部署
- **修复过程**：
  1. 修正 `.nvmrc` 格式：`nodejs_version=22` → `22`
  2. `package.json` 添加 `engines.node >=18.0.0`
  3. `vercel.json` 加回显式 `framework`/`buildCommand`/`outputDirectory`
  4. **用户操作**：Vercel Dashboard → Settings → Build & Deployment → Root Directory → 清空 `"图片百宝箱"` → Save → Redeploy
- **结果**：构建成功，新网页上线
- **Commit**：1ced3ed（最终）
